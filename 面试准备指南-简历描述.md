**项目描述**:

- 使用 **Alamofire** 封装网络请求层，实现统一的错误处理和请求管理
- 采用 **LazyVStack** 优化列表性能，使用 **SDWebImage** 实现图片异步加载和缓存

**项目成果**:
- 代码量 2000+ 行，实现 8 个主要功能模块
- 列表滚动帧率稳定在 60fps，图片加载速度提升 40%

### 版本 2: 详细版（适合面试时展开讲解）

**仿微博社交媒体应用**  
**技术栈**: SwiftUI、SwiftData、Combine、Alamofire、AVKit、CocoaPods  
**项目时间**: 2026.01（持续迭代中）  
**GitHub**: [项目链接]  

**项目背景**:  
为了深入学习 iOS 17 的新特性和 SwiftUI 开发，独立设计并实现了一个功能完整的社交媒体应用，涵盖了帖子发布、多媒体展示、社交互动等核心功能。

**技术架构**:
- **UI 层**: SwiftUI 声明式 UI，MVVM 架构模式
- **数据层**: SwiftData 持久化存储，支持 JSON 导入导出
- **网络层**: Alamofire + NetworkManager 单例模式
- **业务层**: JSONService 统一管理数据同步

**核心功能实现**:

1. **帖子列表与展示**
   - 使用 `@Query` 实现数据自动同步，支持推荐/热门分类展示
   - 采用 `LazyVStack` 实现列表懒加载，优化内存占用
   - 实现下拉刷新和上拉加载更多，支持分页加载

2. **多图智能布局**
   - 设计并实现 1-6 张图片的自适应布局算法
   - 使用 `GeometryReader` 动态计算图片尺寸
   - 集成 JXPhotoBrowser 实现全屏图片浏览，支持缩放、拖拽手势

3. **视频播放功能**
   - 使用 AVKit 的 `VideoPlayer` 实现视频播放
   - 通过 Combine 监听播放器状态，实现自动播放和循环播放
   - 自定义静音按钮和全屏播放功能，优化用户体验
   - 实现页面消失时自动释放播放器资源，避免内存泄漏

4. **数据持久化**
   - 使用 `@Model` 宏定义数据模型，实现类型安全的数据存储
   - 设计 JSON 与 SwiftData 的双向转换机制
   - 图片采用文件名存储方案，减小数据库体积 90%
   - 实现数据的增删改查和实时同步

5. **网络请求架构**
   - 封装 NetworkManager 单例，统一管理网络请求
   - 实现泛型数据解析，支持类型安全的 JSON 转模型
   - 设计友好的错误处理机制，区分网络错误、解析错误和业务错误
   - 配置统一的请求头和超时时间

6. **性能优化**
   - 使用 SDWebImage 实现图片异步加载和三级缓存（内存/磁盘/网络）
   - 通过 `@Bindable` 实现精确的视图更新，避免不必要的重绘
   - 使用 Predicate 优化数据库查询，减少内存占用
   - 实现视频按需加载和自动释放机制

**技术难点与解决方案**:

1. **SwiftData 数据同步**
   - **难点**: 如何实现 JSON 文件与 SwiftData 的双向同步
   - **方案**: 设计 `convenience init?(from:)` 和 `toJSON()` 方法，实现数据的导入导出
   - **优化**: 使用 `@Attribute(.unique)` 确保 ID 唯一性，避免重复数据

2. **视频播放器状态管理**
   - **难点**: 自定义静音按钮与系统控制栏冲突
   - **方案**: 使用 Combine 监听 `isMuted` 属性变化，实现双向绑定
   - **优化**: 使用 `@StateObject` 管理播放器状态，避免状态丢失

3. **图片浏览器索引定位**
   - **难点**: 点击不同图片时浏览器总是显示第一张
   - **方案**: 使用 `.id()` 修饰符强制重新创建视图
   - **优化**: 延迟显示浏览器，确保状态更新完成

4. **列表滚动性能**
   - **难点**: 100+ 条帖子时滚动卡顿
   - **方案**: 使用 `LazyVStack` 替代 `VStack`，实现按需加载
   - **优化**: 图片异步加载，视频懒加载，减少内存峰值

**项目成果**:
- 完成 2000+ 行代码，实现 15+ 个可复用组件
- 列表滚动帧率从 30fps 提升到 60fps
- 图片加载速度提升 40%，内存占用降低 50%
- 掌握了 SwiftUI、SwiftData、Combine 等现代 iOS 开发技术

**后续规划**:
- 添加评论功能，实现 Post 和 Comment 的一对多关系
- 实现用户认证和个人主页功能
- 添加单元测试，提高代码质量
- 对接真实的后端 API
